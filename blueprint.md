# ONLYPANTS | Official Store Blueprint

## 1. 프로젝트 개요

"ONLYPANTS" 브랜드를 위한 인터랙티브하고 모던한 온라인 스토어 웹사이트입니다. 사용자에게 독특한 경험을 제공하기 위해 인트로 애니메이션과 동적 상품 목록을 특징으로 합니다.

---

## 2. 디자인 및 기능 명세

### 2.1. 스타일 및 디자인

*   **색상 팔레트:**
    *   `--blue: #00aff0` (주요 색상)
    *   `--gray: #f5f5f5` (배경 등)
    *   `--text: #111` (기본 텍스트)
*   **폰트:** Pretendard
*   **레이아웃:**
    *   Flexbox와 Grid를 사용한 반응형 레이아웃.
    *   모바일에서는 상품 목록이 2열, 데스크톱에서는 5열로 표시됩니다.
*   **애니메이션:**
    *   페이지 전환 시 부드러운 `opacity` 전환 효과.
    *   인트로 버튼에 시선을 끄는 `glowing` 애니메이션.
    *   상품 이미지에 마우스를 올리면 이미지가 확대되는 효과.

### 2.2. 핵심 기능

*   **인트로 페이지:**
    *   `bg.jpg` 배경 이미지를 보여주는 초기 화면.
    *   "온리팬츠 입으러 가기" 버튼을 누르면 비디오 섹션으로 전환됩니다.
*   **비디오 섹션:**
    *   `video.mp4` 영상을 자동 재생합니다.
    *   영상 재생이 끝나면 자동으로 쇼핑몰 페이지로 전환됩니다.
    *   영상이 5초 이상 재생되지 않을 경우를 대비해 안전장치가 마련되어 있습니다.
*   **쇼핑몰 페이지:**
    *   상단에 로고가 있는 헤더가 고정되어 있습니다. (로고 클릭 시 새로고침)
    *   30개의 상품이 동적으로 생성되어 그리드 형태로 표시됩니다.
    *   각 상품은 브랜드명, 상품명, 할인가, 원가를 포함합니다.

## 4. 프리미엄 강화 계획 (Planned)

### 4.1. 목표
`GEMINI.md` 가이드라인에 따라 시각적 완성도를 높이고, 최신 웹 기술을 활용하여 아키텍처를 개선합니다. "Premium, Tactile, Vibrant"한 사용자 경험을 제공하는 것이 목표입니다.

### 4.2. 실행 단계

*   **1단계: 시각적 스타일 고도화 (CSS)**
    *   **텍스처 적용:** 메인 배경에 미세한 노이즈 텍스처를 추가하여 고급스러운 질감을 부여합니다.
    *   **깊이감 강화:** 상품 카드에 다층 드롭 섀도우(Multi-layered drop shadows)를 적용하여 "떠 있는" 듯한 효과를 줍니다.
    *   **컬러 및 타이포그래피:** `oklch()` 컬러 공간을 활용하여 더 생생한 블루 포인트를 적용하고, 서체 강조(Hero text, Section headlines)를 통해 가독성을 개선합니다.
    *   **Cascade Layers (`@layer`):** CSS 구조를 초기화(base), 레이아웃(layout), 컴포넌트(components), 유틸리티(utils) 레이어로 분리하여 관리합니다.

*   **2단계: 웹 컴포넌트 도입 (JS)**
    *   **`<pant-item>` 컴포넌트 생성:** 각 상품 항목을 독립적인 Web Component로 캡슐화합니다.
    *   **Shadow DOM 활용:** 컴포넌트 내부 스타일과 구조를 보호합니다.
    *   **동적 데이터 바인딩:** 다양한 상품 정보를 속성(Attributes)을 통해 전달받아 렌더링합니다.

*   **3단계: 인터랙션 및 콘텐츠 개선**
    *   **호버 효과 정교화:** 상품 호버 시 단순 확대 외에 추가 정보(예: "Quick View")가 나타나도록 개선합니다.
    *   **데이터 다양화:** 30개의 동일한 상품 대신, 다양한 이름과 가격대를 가진 상품 데이터를 생성하여 실제 스토어처럼 구성합니다.
    *   **반응형 최적화:** Container Queries를 사용하여 부모 컨테이너 크기에 따라 컴포넌트 내부 레이아웃이 최적으로 조절되도록 합니다.
