# ONLYPANTS | Official Store Blueprint

## 1. 프로젝트 개요

"ONLYPANTS" 브랜드를 위한 인터랙티브하고 모던한 온라인 스토어 웹사이트입니다. 사용자에게 독특한 경험을 제공하기 위해 인트로 애니메이션과 동적 상품 목록을 특징으로 합니다.

---

## 2. 디자인 및 기능 명세

### 2.1. 스타일 및 디자인

*   **색상 팔레트:**
    *   `--blue: #00aff0` (주요 색상)
    *   **Light Mode:** White background, dark text.
    *   **Dark Mode:** Dark background (#111), light text (#f5f5f5).
*   **폰트:** Pretendard
*   **레이아웃:**
    *   Flexbox와 Grid를 사용한 반응형 레이아웃.
    *   모바일에서는 상품 목록이 2열, 데스크톱에서는 5열로 표시됩니다.
*   **애니메이션:**
    *   페이지 전환 시 부드러운 `opacity` 전환 효과.
    *   인트로 버튼에 시선을 끄는 `glowing` 애니메이션.
    *   상품 이미지에 마우스를 올리면 이미지가 확대되는 효과.
    *   **프리미엄 엔트리 효과:** 상품 렌더링 시 Staggered Blur-to-Focus 애니메이션 적용.

### 2.2. 핵심 기능

*   **인트로 페이지:**
    *   `bg.jpg` 배경 이미지를 보여주는 초기 화면.
    *   "온리팬츠 입으러 가기" 버튼을 누르면 비디오 섹션으로 전환됩니다.
*   **비디오 섹션:**
    *   `video.mp4` 영상을 자동 재생합니다.
    *   영상 재생이 끝나면 자동으로 쇼핑몰 페이지로 전환됩니다.
*   **쇼핑몰 페이지:**
    *   상단 헤더에 로고와 **다크/라이트 모드 토글 스위치**가 포함되어 있습니다.
    *   30개의 상품이 동적으로 생성되어 그리드 형태로 표시됩니다.
    *   **테마 지속성:** 사용자가 선택한 테마는 `localStorage`에 저장되어 재방문 시에도 유지됩니다.

## 4. 프리미엄 강화 계획 (Planned)

### 4.1. 목표
`GEMINI.md` 가이드라인에 따라 시각적 완성도를 높이고, 최신 웹 기술을 활용하여 아키텍처를 개선합니다. "Premium, Tactile, Vibrant"한 사용자 경험을 제공하는 것이 목표입니다.

### 4.2. 실행 단계

*   **1단계: 시각적 스타일 및 인터랙션 고도화 (CSS/JS) [IN PROGRESS]**
    *   **다크/라이트 모드 지원:** 슬라이드 토글 버튼을 통해 사용자가 원하는 테마를 선택할 수 있도록 구현합니다. (완료)
    *   **프리미엄 이미지 엔트리 효과:** 첫 상품 렌더링 시 세련된 애니메이션(Blur-to-Focus, Staggered Slide-up)을 적용합니다. (완료)
    *   **텍스처 적용:** 메인 배경에 미세한 노이즈 텍스처를 추가하여 고급스러운 질감을 부여합니다. (완료)
    *   **깊이감 강화:** 상품 카드에 다층 드롭 섀도우(Multi-layered drop shadows)를 적용합니다.
    *   **컬러 및 타이포그래피:** `oklch()` 컬러 공간을 활용하여 더 생생한 블루 포인트를 적용합니다.

*   **2단계: 웹 컴포넌트 도입 (JS)**
    *   **`<pant-item>` 컴포넌트 생성:** 각 상품 항목을 독립적인 Web Component로 캡슐화합니다.
    *   **Shadow DOM 활용:** 컴포넌트 내부 스타일과 구조를 보호합니다.

*   **3단계: 인터랙션 및 콘텐츠 개선**
    *   **호버 효과 정교화:** 상품 호버 시 추가 정보가 나타나도록 개선합니다.
    *   **반응형 최적화:** Container Queries를 사용하여 레이아웃을 최적화합니다.
